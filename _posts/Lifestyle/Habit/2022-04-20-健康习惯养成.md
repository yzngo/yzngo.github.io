## 食

### 肉蛋

- [x] 多吃白肉，少吃红肉
- [x] 少吃蛋黄

### 素食

- [x] 多吃蔬菜，多吃辣
- [x] 多吃坚果
- [x] 多吃植物蛋白
- [x] 少吃碳水
- [x] 吃复合维生素

### 饮品

- [x] 多喝咖啡，牛奶，茶
- [x] 少喝含糖饮料

## 动

- [x] 每 20 分钟变换一次姿势（站 -坐 - 站 - 坐 - 站）
- [x] 每小时至少走动 5 分钟

## 睡

- [x] 22-24 点间睡觉，每天睡 7 小时

## 日常

- [x] **每天晒太阳 10 分钟**
- [x] 每天刷牙
