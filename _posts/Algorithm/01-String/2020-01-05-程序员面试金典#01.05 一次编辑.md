## 题目

There are three types of edits that can be performed on strings: insert a character, remove a character, or replace a character. Given two strings, write a function to check if they are one edit (or zero edits) away.

**Example 1:**

```
Input: 
first = "pale"
second = "ple"
Output: True
```

**Example 2:**

```
Input: 
first = "pales"
second = "pal"
Output: False
```

<br>

## 题解

1. 分别从前往后，从后往前遍历一次，找到不相同的字符区间，如果此字符区间只有一个字符，则满足条件。 $T = O(n)$

    ```c++
    bool oneEditAway(string first, string second) {
        if (first.length() < second.length()) {
            swap(first, second);
        }
        
        if (first.length() == 1 || first.length() == 0) return true;
        if (first.length() - second.length() >= 2) return false;
        
        int i = 0;
        for(; i != second.length(); ++i) {
            if (first[i] != second[i]) {
                break;
            }
        }
        
        int j = (int)first.length() - 1;
        int k = (int)second.length() - 1;
        for(; j >= 0 && k >= 0; --j, --k) {
            if (first[j] != second[k]) {
                break;
            }
        }
        return j - i <= 0 ;
    }
    ```

2. 模拟删除和添加过程。此解法没有方法1快，但比较直观。

    ```c++
    bool oneEditAway(string first, string second) {
        int len = first.length() - second.length();
        if (len > 1 || len < -1) {
            return false;
        }
        
        int count = 1;
        for (int i = 0, j = 0; i < first.length() && j < second.length(); ++i, ++j) {
            if (first[i] != second[j]) {
                if (len == 1) {		//second要不要添加一个字符
                    j--;
                }else if ( len == -1) {		//second要不要删除一个字符
                    i--;
                }
                count--;
            }
            if (count < 0) {	//最多编辑一次
                return false;
            }
        }
        return true;
    }
    ```
