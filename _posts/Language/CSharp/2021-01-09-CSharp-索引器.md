## 索引器 Indexers

1.  [微软文档](https://docs.microsoft.com/zh-cn/dotnet/csharp/programming-guide/indexers/)

1. 使类或结构的实例可以使用数组一样的索引方式。

   ```c#
   private T[] arr = new T[100];
   public T this[int i]
   {
      get { return arr[i]; }
      set { arr[i] = value; }
   }
   ```
