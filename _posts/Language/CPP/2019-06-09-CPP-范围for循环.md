## for range

```c++
string str = "hello world";
// read only
for (auto c : str)
{
    std::cout << c << std::endl;
}

// 如果想要改变 string 对象中的值，必须把循环变量定义成引用类型(只是起个别名)。
for (auto &c : str) {
    c = tou
}
```
