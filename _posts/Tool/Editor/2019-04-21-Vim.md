## 行内导航


| 行内导航            | Key   |
| ------------------- | ----- |
| 跳到行首            | 0     |
| 跳到行尾            | $     |
| 上一个单词头        | b / B |
| 下一个单词头        | w / W |
| 下一个单词尾        | e / E |
| 下一个为 x 的字符   | fx    |
| 上一个为 x 的字符   | Fx    |
| 下一个 f搜索的结果  | ;     |
| 上一个 f 搜索的结果 | ,     |
