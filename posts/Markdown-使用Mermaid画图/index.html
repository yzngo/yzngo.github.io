Markdown-使用Mermaid画图

By Yzngo
Posted 2019-04-23 Updated 2022-06-05 2 min read

Mermaid 是一个基于 Javascript 的图表绘制工具，通过解析类 Markdown 的文本语法来实现图表的创建和动态修改。

如果使用Jekyll发布，FrontMatter中需要添加 mermaid: true

	B --> C{Decision}
	C -->|One| D[Result 1]
	C -->|Two| E[Result 2]
